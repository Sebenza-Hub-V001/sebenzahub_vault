# Welcome to the Sebenza Hub Wiki

This is the knowledge base for **Sebenza Hub** — a multi-sided recruitment and career platform built for the South African employment market.

## What is Sebenza Hub?

Sebenza Hub connects four user types through a unified platform:

| User Type | What They Do |
|-----------|-------------|
| **Individuals** | Find jobs, build skills, manage their career |
| **Recruiters** | Source candidates, manage clients, earn placement fees |
| **Businesses** | Post jobs, hire directly, manage workforce |
| **Admins** | Configure the platform, monitor health, enforce compliance |

For the full picture, start with the [[overview]].

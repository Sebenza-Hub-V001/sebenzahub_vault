# Source: Individual Features Guide

**Raw file:** `raw/articles/individual-features.md`
**Ingested:** 2026-04-07

## Summary

A comprehensive guide detailing every feature available to Individual users (job seekers) on Sebenza Hub, structured as "what it is / how to use it / why it matters" for each feature. Covers 56+ features across Profile, Job Search, Applications, AI Tools, Learning Hub, Career Development, and Community.

## Key Claims

### Profile & Identity (5 features)
- **My CVs** — Multi-version CV management, tailored to different industries
- **Credentials** — Verified digital wallet for qualifications with trust badges and expiry reminders
- **Video Profile** — 60–90 second video introduction visible to recruiters
- **Skill Verifications** — Standardised tests that add verified badges to skills
- **CV Templates** — Library of professionally designed templates

### Job Search (5 features)
- **All Jobs** — Centralised job board with multi-filter search
- **Auto Match** — AI-powered continuous job matching with notifications
- **Manual Search** — Advanced filters (location, salary, industry, remote/hybrid/onsite)
- **Saved Searches** — Persistent search criteria with new-match alerts
- **Favourites** — Personal job shortlist for comparison

### Applications (7 features)
- **My Applications** — Dashboard of all applications with status tracking
- **App Tracker** — Visual timeline of each application's progress
- **App Analytics** — Personal success metrics (response rates, conversion)
- **Notes & Tags** — Personal CRM for job applications
- **Reminders** — Customisable alerts for follow-ups and deadlines
- **Application Portal** — Unified portal for employer requests
- **Candidate Hub** — Aggregated dashboard of priorities and actions

### AI Tools (4 features)
- **AI Coaching** — On-demand personalised career guidance
- **CV Review** — Detailed CV analysis with actionable feedback
- **Cover Letter** — AI-generated role-specific cover letters
- **Salary Negotiator** — Data-driven negotiation prep with market data

### Learning Hub (12 features)
- **Learning Dashboard, Paths, Skill Tree, Micro Learning, Interview Simulator**
- **Job-Linked Learning** — Skills gap courses tied to specific job listings
- **Challenges** — Project-based assessments that become portfolio pieces
- **Portfolio Builder** — Showcase work, projects, and achievements
- **Peer Teaching** — Teach skills you've mastered, learn from others
- **SETA Learnerships** — SA-specific accredited learnership programmes
- **Certifications, Study Planner, Learning Analytics, Mentorship**

### Career Development (6 features)
- **Skill Gap, Career Path, Salary Insights, Career Pulse, Tests, Analytics**

### Community (4 features)
- **Forum, Trending, Leaderboard, Badges**
